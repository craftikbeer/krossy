# Каталог кроссовок деревня.com

## Описание проекта
Интернет-каталог кроссовок с интеграцией Telegram для связи с клиентами. Вместо традиционной корзины и оформления заказа, покупатели связываются с продавцом напрямую через Telegram.

## Основные функции
- Элегантная hero-секция с названием магазина
- Просмотр каталога кроссовок
- Фильтрация по брендам
- Поиск по названию модели
- Кнопка связи через Telegram на каждой карточке товара
- Адаптивный дизайн для всех устройств

## Структура проекта
- `index.html` - главная страница
- `app.js` - основная логика приложения
- `components/Header.js` - шапка сайта
- `components/Hero.js` - hero-секция с названием магазина
- `components/SneakerCard.js` - карточка товара
- `components/FilterBar.js` - панель фильтров и поиска

## Настройка
Замените `https://t.me/yourusername` в файлах `Header.js` и `SneakerCard.js` на ваш реальный Telegram username.

## Технологии
- React 18
- Tailwind CSS
- Lucide Icons
